\cventry{2025.06--2025.09}{大模型应用工程师}{武汉文弘文创}{智能客服Agent系统}{}{%
  项目介绍: 针对“步道乐跑”APP数千万大学生用户的咨询需求(如跑量异常、打卡故障等)，传统客服依赖电子手册存在定位慢、效率低、易出错等痛点。为降低人力成本并实现7×24小时精准服务，独立开发了集成RAG和自动化工具链的智能客服Agent系统。\\
  项目主要内容:\\
  1. RAG知识库深度优化: 针对非结构化《客服操作手册》，采用父子分块策略。父块保留上下文语义，子块细化具体操作步骤。解决了大模型在处理长文本时“幻觉”严重问题，将业务知识的检索召回率提升了30\%，确保回答严格符合业务准则，通过流程管控将高风险误答率控制在极低水平。\\
  2. Agent架构: 利用LangChain框架，结合ReAct推理模式与有限状态机(FSM)。FSM负责锁定并管控高风险业务流程(如数据库查询等)，防止越权或错误执行，ReAct负责灵活处理用户多样化的咨询意图。\\
  3. MCP Server工具化封装与动态编排: 将后端数据库查询、打卡记录修复、日志提取等工具封装为标准MCP服务。支持根据用户意图，通过异步/并发策略动态调用多个工具，并对工具调用设定权限边界与输入检查。\\
  *项目价值：该项目深入实践了智能体工具调用与流程编排，为研究智能体在复杂交互中的行为安全、风险识别与管控机制提供了工程基础。*
}
\cventry{2024.09--2025.06}{}{}{铸牢中华民族共同体知识大模型对话系统}{}{%
  项目介绍: 基于大语言模型构建智能对话系统，通过权威政策和历史文化数据微调模型，结合RAG和Agent技术，实现多模态交互与复杂任务编排，提供民族领域智能知识服务，并重点保障生成内容的\textbf{事实性、权威性与安全性}。\\
  核心工作:\\
  1. 数据集构建: 基于多源异构数据采集和精细化清洗流程，构建了涵盖学术文献、政策文件及官方报道的领域专属语料库，并通过分层提示词和自生成与增强生成双路径机制，形成了包含发散性、专业性、权威性三层次的约17万条高质量监督微调数据集。\\
  2. 模型微调: 采用“继续预训练知识注入”和“递进式监督微调”，首先利用构建的领域语料库对基座模型进行继续预训练以注入领域知识，随后将微调数据集按发散性、专业性、权威性进行层次化划分，并基于LoRA技术依次递进微调。最终模型在领域客观任务上较基座模型提升20.4\%，有效解决了领域知识幻觉与语言理解不深入等难题，提升了模型输出的可靠性与可控性。\\
  3. RAG对话模块: 构建了面向垂直领域的自适应双层检索增强生成架构，引入DeepSeek-V3.2作为意图判别路由，采用BGE-M3嵌入模型进行稠密向量化，基于FAISS向量数据库实现关键词聚类分簇存储，通过“关键词匹配限定检索范围+簇内稠密向量相似度计算”的双层检索机制，将Top-5召回准确率提升至87.5\%，MRR达0.734，有效降低了模型生成的事实性幻觉，为内容安全与可信生成提供了技术保障。\\
  *项目价值：通过全链路优化，探索了在垂直领域约束下提升大模型生成事实准确性与可靠性的方法，涉及数据、训练与推理环节，与研究导向的技术预研紧密结合。*
}
